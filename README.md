# IPL-Team-Performance-Analysis
I’m excited to share BI project based on IPL Winning data. It includes, 📊 Year-wise and team-wise match results and win trends 🧢 Innings wise Winning Count 🏏 Year wise no. of runs, no. of boundaries, wickets put on board. 📍 Venue-wise match outcomes and team performance 📈 Visual storytelling with slicers, cards, maps, bar and Matrix Charts.
